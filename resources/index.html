Leo OS principle

Useful tools should make the next right action obvious. If a resource makes people admire the idea but not use it, it is still too fancy.

Standard: clarity first, specificity second, visible action third.

Use this hub for

  • Meeting planning: theme, timing, roles, risks, and handoffs.
  • Speech building: structure, delivery, evaluation, and promo.
  • Club PR: stories, social posts, newsletters, and guest invitations.
  • Officer support: communication, scheduling, onboarding, and follow-through.

The resource rule

Every resource should pass the “Monday night test”: could a club officer, speaker, or evaluator actually use this at the next meeting?

Clear Easy to understand fast.
Specific Built for a real use case.
Actionable Produces a next step.
Reusable Works more than once.
